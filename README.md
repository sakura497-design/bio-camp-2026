# 2026 生物相关保研夏令营信息看板

从 [shenyanpai/awesome-summer-camp-2026](https://github.com/shenyanpai/awesome-summer-camp-2026) 仓库自动抓取并筛选生物相关夏令营信息。

## 快速开始

```bash
npm install
npm run dev      # 开发 → http://localhost:3000
npm run build    # 构建 → dist/
npm run test     # 测试数据解析
```

## 功能

- 自动从 GitHub 仓库抓取最新 Markdown 数据
- 三级关键词过滤，只保留生物相关条目
- 搜索、筛选（学校/截止日期/来源）、排序
- 收藏重点条目
- 深色/浅色模式
- 手机和桌面端适配

## 技术栈

React 19 + TypeScript + Tailwind CSS 4 + Vite 8 + Lucide Icons

## 数据来源

夏令营通知数据来自 [shenyanpai/awesome-summer-camp-2026](https://github.com/shenyanpai/awesome-summer-camp-2026)（MIT License），由「深研派」团队整理。本项目仅做数据筛选和展示，不拥有原始数据的版权。
